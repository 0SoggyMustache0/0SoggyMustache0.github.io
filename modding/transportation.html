Transportation Mod

This tutorial will teach you how to make your own transportation mod

For this tutorial the mod will be called george's transportation mod but you can call it whatever you want

Step: 1

First we need to download the example source code for the mod HERE

Step: 2

Now we need to download the minecraft forge 1.9 MDK HERE I used version 1.9 - 12.16.1.1887

Step: 3

Now we need to setup the workspace like a normal mod if you don't know how to do this go to the other tutorials before this one.

Step: 4

Create a new folder in the mod folder called libs and place the latest version of moon's core and soggymustache's transportation mod 1.9 in the folder then open the command window in the main folder and do 'gradlew cleanEclipse' and then 'gradlew eclipse' if that was confusing watch this.

Step: 5

Now that all the hard stuff is done copy the source code from my example and place it in your folder and open up eclipse or what ever you are using to code the mod

Step: 6

You can now go and change the reference, and mcmod.info file to your liking, after that just look in any of the example code and it will guide you through making the vehicles work

Help:
