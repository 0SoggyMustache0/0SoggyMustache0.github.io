Achievements

This tutorial will teach you how to make simple achievements

First we need to register the achievement class in out common proxy 
public class CommonProxy {
            public void init(FMLInitializationEvent event)	
	           {
		TutorialAchievement.init(event);
	           }
    }
The 'TutorialAchievement' is going to be in red so hover over it to create the class 
          
package net.soggymustache.tutorial.init.achieve;

import net.minecraft.init.Items;
import net.minecraft.stats.Achievement;
import net.minecraftforge.common.AchievementPage;
import net.minecraftforge.fml.common.event.FMLInitializationEvent;


public class TransportationAchievement {
	
   //this is where the new achievements are started
    public static Achievement achievementCraftPumpkinPie;
    
    
	public static void init(FMLInitializationEvent event)
	{
        //this is where all the achievements get their info                              //.setSpecial gives the achivement a different look to stand out>
//we can make the achivement need another before we can get it by changing (Achievement)null) to the achievement we want>
        achievementCraftPumpkinPie = new Achievement("achievement.CraftPumpkinP", "PumpkinP", 1, -2, Items.pumpking_pie, (Achievement)null).registerStat().setSpecial;

        //this is where the achievements are registered
        AchievementPage.registerAchievementPage(new AchievementPage("Tutorial Achievements", new Achievement[]{achievementCraftPumpkinPie}));
	}
}
Now if we run the game our achievements will show up in our achievement page but we can't get them yet

We now need to make a new class called 'CraftingEvent' or something like that

package net.soggymustache.tutorial.init.achieve.event;

import net.minecraftforge.fml.common.eventhandler.SubscribeEvent;
import net.minecraftforge.fml.common.gameevent.PlayerEvent;
import net.soggymustache.tutorial.init.achieve.TutorialAchievement;
import net.minecraft.init.Items;

public class CraftingEvent {
	
	@SubscribeEvent
    //instead of ItemCraftedEvent we can do ItemSmeltedEvent for when the player smelts something
	public void itemCrafted(PlayerEvent.ItemCraftedEvent e)
	{
		if(e.crafting.getItem().equals(Items.pumpkin_pie))
		{
			e.player.addStat(TutorialAchievement.achievementCraftPumpkinPie);
		}
	}
	

}
Now that class will work but we need to register it 
public class CommonProxy {
	public void preInit(FMLPreInitializationEvent event)
	{
		FMLCommonHandler.instance().bus().register(new CraftingEvent());
	}
    }
Now we can run the game craft a pumpkin pie and get an achievement
