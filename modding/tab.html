Creative Tabs

This tutorial will teach you how to make creative tabs

The first bits of code will go in your main class

The part in blue is what you need (I didnt include all the code of the main class because there is too much)

package net.soggymustache.tutorial;

@Mod(modid = Reference.MOD_ID, name = Reference.MOD_NAME, version = Reference.VERSION)
public class TutorialMain {
	
	@SidedProxy(clientSide = Reference.CLIENT_PROXY_CLASS, serverSide = Reference.SERVER_PROXY_CLASS)
	public static CommonProxy proxy;
	
	public static final TutorialTab tabTutorial = new TutorialTab("tabTutorial");
	
	@Instance("tutorial")
	public static TutorialMain modInstance;
	
	@EventHandler
	public void preInit(FMLPreInitializationEvent e)
	{		
		proxy.preInit(e);
	}
}

TutorialTab is going to be in red since we havent made that class yet
Hover over 'TutorialTab' and create the class

package net.soggymustache.tutorial.tab;

import net.minecraft.creativetab.CreativeTabs;
import net.minecraft.init.Blocks;
import net.minecraft.item.Item;
import net.soggymustache.tutorial.item.TutorialItem;

public class TutorialTab extends CreativeTabs{

	public TutorialTab(String label) {
		super(label);
	}

	@Override
	public Item getTabIconItem() {
		return Item.getItemFromBlock(Blocks.dragon_egg);
	}

}
